Scenario: Authentication with AcademicID

Prerequisites: To complete this test case you will need to download the Gataca wallet and provision your credentials. For the provisioning go to https://dss.aegean.gr/erua-issuer and follow the steps from this guide.

In this variant the AcademicID credential is used. The steps to be carried out are illustrated in the following diagram.

To summarise:

  1. The test engine will trigger a credential request to the Verifier Service for an AcademicID.
  2. You will need to open your wallet application and scan the resulting QR code to confirm.
  3. The test engine will finally check the Verifier Service for the expected validation response.
